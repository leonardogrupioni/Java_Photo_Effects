
 ============================================ README =====================================
 O Processador de Imagens é uma aplicação em Java que tem o intuito de aplicar filtros  em 
 imagens JPEG e gerar novos arquivos de imagem, além de permitir a visualização das mesmas
 e de  um Histograma (gráfico que permite a visualização da quantidade de  pixels com  uma
 determinada cor) da imagem em tons de cinza. 
 =========================================================================================

 TITULO DO PROJETO: PROCESSADOR DE IMAGENS EM JAVA
 OBJETIVO DO PROJETO: Receber uma imagem em formato JPEG e gerar outras 4 imagens apli-
                      cando filtros (Preto e Branco, Tons de Cinza, Sepia e Invertido),
                      permitindo a visualização delas e o salvamento em qualquer lugar
                      do computador. Além disso, permite a visualização de um Histograma 
                      da imagem com o filtro Tons de Cinza aplicado.
 VERSAO OU DATA: 20/11/2023 - v1.0
 COMO INICIAR O PROJETO: Para iniciar o projeto abre o arquivo Package, em seguida no
                         no BlueJ clique com o botão direito na classe app e em seguida
                         clique sobre o método void main e clique em Ok.
 AUTORES: João Pedro Figols Neco, Júlia Gachido Schmidt e Leonardo Fajardo Grupioni.
 INSTRUÇÕES PARA O USUÁRIO: Ao iniciar a aplicação carregue uma imagem no formato JPEG
                            (.jpg) em seguida você pode visualizar todas as imagens com
                            os filtros aplicados e também o histograma da imagem com o
                            filtro Tons de Cinza, por fim caso queira salvar as imagens
                            clique em Salvar e selecione onde deseja que os arquivos 
                            sejam criados.
